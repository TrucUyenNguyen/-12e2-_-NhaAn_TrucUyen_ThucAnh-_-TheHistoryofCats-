## Chủ đề dự án

Website trình bày cách con người mô tả loài mèo qua các thời kỳ lịch sử: **Ai Cập cổ đại – Trung Cổ – Hiện Đại**.
Dựa trên W3.CSS, nhóm tùy biến template, thay đổi màu sắc, hình ảnh và bố cục để website mang phong cách của một “bảo tàng trực tuyến về mèo”.
